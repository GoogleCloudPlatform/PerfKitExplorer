Cloud SQL Settings
Provide the credentials used to authenticate to Cloud SQL datasources. Database owners will need to grant permissions to this account. Ensure that you only grant SELECT access to the relevant tables/views. You can read more about cloudsql security in the MySql documentation, or use the example scripts below.
CREATE USER '{{ ngModel.username || 'username' }}'@'localhost' IDENTIFIED BY '{{ ngModel.password || 'password' }}';
GRANT SELECT ON db1.* TO '{{ ngModel.username }}'@'localhost';
GRANT SELECT ON db2.table1 TO '{{ ngModel.username }}'@'localhost';
